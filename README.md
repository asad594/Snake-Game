# Snake-Game-
🐍 Snake Game – Python GUI (OOP Based)

A fully functional Snake Game built using Python, following Object-Oriented Programming (OOP) principles and a graphical user interface.
This project demonstrates clean code structure, game logic implementation, and GUI-based event handling.

🚀 Features

🎮 Classic Snake gameplay

🧠 Object-Oriented Design (separate classes for game logic, snake, food, and UI)

🖥️ Interactive GUI

⌨️ Keyboard controls

📈 Score tracking

🔁 Game over & restart mechanism

⚡ Smooth movement and collision detection

🛠️ Technologies Used

Python

OOP (Object-Oriented Programming)

GUI Framework (Tkinter / Pygame – update based on what you used)

Event Handling

Game Loop & Collision Logic

🧩 Project Structure
Snake-Game/
│
├── main.py          # Entry point of the game
├── snake.py        # Snake class (movement & growth)
├── food.py         # Food generation logic
├── scoreboard.py   # Score handling
├── game.py         # Core game logic
└── README.md


(Modify filenames if yours are different)

🎮 How to Play

Arrow Keys → Move the snake

Eat food → Grow the snake and increase score

Hit wall or self → Game Over

▶️ How to Run the Project

Clone the repository

git clone https://github.com/your-username/snake-game-python.git


Navigate to the project folder

cd snake-game-python


Run the game

python main.py
